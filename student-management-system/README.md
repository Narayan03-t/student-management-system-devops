# Student Management System (DevOps Project)

## Project Overview
This is a Java-based Student Management System developed using Maven and Git as part of the DevOps course.

The project demonstrates:
- Maven project structure
- Dependency management
- Git version control
- GitHub hosting

## Features
- Add student details
- Display student records
- Academic data tracking:
  - Department
  - CGPA
  - Attendance
  - Backlogs
  - Assignment status

## Technologies Used
- Java
- Maven
- Git
- GitHub
- Eclipse IDE

## Project Structure
- model: Student entity
- dao: Data access layer
- repository: Storage logic
- service: Business logic
- app: Main application



## ⚙️ Maven Lifecycle
Maven follows these phases:

1. validate
2. compile
3. test
4. package
5. install
6. deploy

Command used:
bash
mvn clean install



## How to Run
1. Import project into Eclipse
2. Run `StudentManagementApp.java`
